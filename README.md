# 블로그 사이트 만들기
### 공용 블로그 누구나 게시글을 올릴 수 있으며 목적은 다양합니다. 
### 맛집 공유, 동네 질문, 동네 소식, 동물 구조  


***

## 페이지, 기능

* 로그인, 회원가입
  * 일반유저 
  * 카카오API


* 게시글 올리기
  * 게시글 작성,수정,삭제 


* 댓글 
  *  댓글 작성,삭제



## 기술 블로그 링크
